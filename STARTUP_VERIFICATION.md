## ⚠️ 可能的问题

### 启动问题

**问题1: 依赖缺失**
```bash
# 解决: 重新安装依赖
npm install
```

**问题2: 端口占用**
```bash
# 解决: 检查5173端口
netstat -ano | findstr 5173

# 杀死占用进程
taskkill /PID <PID> /F
```

**问题3: TypeScript编译错误**
```bash
# 解决: 清理并重新构建
npm run build:main
```

### LSP问题

**问题1: 语言服务器未安装**
```bash
# TypeScript
npm install -g typescript-language-server typescript

# Python
pip install python-lsp-server

# Go
go install golang.org/x/tools/gopls@latest

# Rust
rustup component add rust-analyzer
```

**问题2: LSP未启动**
- 检查控制台是否有LSP启动日志
- 查看StatusBar右侧是否显示LSP状态
- 尝试手动重启LSP

### 调试器问题

**问题1: 调试器无法启动**
- 检查launch配置
- 验证程序路径
- 查看控制台错误

### AI问题

**问题1: API Key未配置**
- 编辑 `src/core/ai/config.ts`
- 添加有效的API Key

**问题2: 网络连接**
- 检查网络代理
- 验证API端点可访问

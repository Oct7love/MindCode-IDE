### Step 1: 启动应用

```bash
npm run dev
```

### Step 2: 打开项目

1. 点击 "Open Folder"
2. 选择你的代码目录

### Step 3: 体验功能

**测试LSP:**
- 打开一个.ts文件
- 输入代码,看到补全提示
- F12跳转定义
- 鼠标悬停查看类型

- 点击侧边栏Debug图标

- 查看变量和调用栈

**测试AI:**
- Ctrl+L打开AI面板
- 输入问题并发送
- 尝试 @file 引用文件
- Ctrl+K内联编辑代码
